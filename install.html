🖥
Windows 电脑
下载安装包，双击即可创建桌面快捷方式
下载 Windows 安装包 直接使用网页版
  1. 点击「下载 Windows 安装包」
  2. 解压下载的 zip 文件
  3. 将「宿舍管理.url」拖到桌面
  4. 双击图标即可打开
提示：Windows 11 用户也可以直接用 Edge 浏览器打开链接，点击「安装应用」按钮。
📱
Android 手机
Chrome 浏览器添加到主屏幕，像原生 App 一样使用
打开应用
  1. 用 Chrome 浏览器打开上面的链接
  2. 点击右上角菜单 ⋮
  3. 选择「添加到主屏幕」或「安装应用」
  4. 桌面出现图标，点击像 App 一样打开
支持离线使用：首次加载后，关闭网络也能打开应用。
🌐

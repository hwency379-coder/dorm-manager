宿舍极简物品管理

安装到桌面，无需每次打开浏览器

Windows 电脑

下载安装包，解压后把 「宿舍管理.url」 放到桌面，双击即可打开。

下载 Windows 安装包
  1. 点击上面的蓝色按钮下载 dorm-manager-desktop.zip
  2. 解压 zip，得到「宿舍管理.url」和「icon.ico」
  3. 把「宿舍管理.url」放到桌面
  4. 双击图标即可打开应用
Android 手机

用 Chrome 浏览器打开链接，添加到主屏幕。

  1. 用 Chrome 打开 https://hwency379-coder.github.io/dorm-manager/
  2. 点击右上角 ⋮ 菜单
  3. 选择「添加到主屏幕」或「安装应用」
  4. 桌面出现图标
iPhone

用 Safari 打开链接，添加到主屏幕。

  1. 用 Safari 打开链接
  2. 点击底部 分享按钮 ↑
  3. 下滑找到「添加到主屏幕」
  4. 点击添加

# 宿舍极简物品管理（Dorm Manager）

一个面向大学生宿舍场景的**个人物品管理工具**，采用极简设计理念，帮助用户对宿舍内所有物品进行功能分区管理、收纳位置标记、使用状态追踪和定期复盘清理。

**核心定位**：每一件东西都有明确功能，该留的留、该丢的丢。

## 🚀 在线使用

👉 **[GitHub Pages 地址]**（部署后自动生效）

直接访问 GitHub Pages 链接即可使用，无需安装任何应用。支持手机浏览器、微信内置浏览器、电脑浏览器访问。

## 📱 功能概览

| 页面 | 功能 |
|------|------|
| **功能页** | 按 8 大分区（衣装、食饮、书籍、杂物、休憩、电子、日用、待处理）浏览物品 |
| **首页** | 地图视图（在平面图上标记收纳点）+ 表格视图（搜索、筛选） |
| **复盘页** | 资产统计、关注预警、温度速查、已处理记录、收纳清单 |

### 数据管理
- **数据导出**：将 all 数据导出为 JSON 备份到本地
- **数据导入**：从 JSON 备份恢复数据
- **微信文本导出**：一键生成格式化清单，可直接粘贴到微信
- **本地存储**：所有数据保存在浏览器 `localStorage`，不上传服务器

## 🏗️ 技术架构

| 项目 | 说明 |
|------|------|
| 前端 | 纯原生 JavaScript（无框架依赖） |
| 样式 | Tailwind CSS（CDN 引入）+ 自定义 CSS |
| 存储 | localStorage（浏览器本地） |
| 部署 | GitHub Pages（免费、自动、可持续） |
| PWA | 支持离线使用、添加到桌面 |

## 🔄 版本管理

本项目使用 Git 进行版本管理，GitHub 自动托管所有历史版本：

- 每次更新只需 `git push`，GitHub Pages 自动重新部署
- Git 历史记录中保留所有旧版本，可随时回滚
- 数据备份（JSON 导出）保存在本地，与代码版本无关

## 🛠️ 本地开发

```bash
# 克隆仓库
git clone https://github.com/你的用户名/dorm-manager.git
cd dorm-manager

# 直接用浏览器打开（无需构建步骤）
open index.html

# 或者启动本地服务器（推荐，用于测试 PWA）
# Python 3
python -m http.server 8080
# Node.js
npx serve .
```

## 📄 数据安全提示

- 所有数据保存在当前浏览器中，更换浏览器/清理缓存会丢失
- **重要操作后请导出 JSON 备份**（设置面板 → 导出 JSON 备份）
- 换设备时通过 JSON 导出/导入迁移数据
